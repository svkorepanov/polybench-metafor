# Gradle installation note

## Why not apt?

The apt package for Gradle on Ubuntu is out of date. Ubuntu ships Gradle 7.4.2
(from 2022), and Gradle 7.x officially supports only up to Java 16. When run
under Java 21, its embedded Groovy and Kotlin runtimes break with:

    Could not open settings generic class cache for settings file 'settings.gradle'
    BUG! exception in phase 'semantic analysis' Unsupported class file major version 65

Gradle does not publish to the Ubuntu apt repositories itself — the package is
maintained by Debian/Ubuntu and lags years behind upstream. Use one of the
options below instead.

## Install Gradle 8

### Option A — manual install

    wget https://services.gradle.org/distributions/gradle-8.13-bin.zip -P /tmp
    unzip /tmp/gradle-8.13-bin.zip -d /opt/gradle
    export PATH=/opt/gradle/gradle-8.13/bin:$PATH
    echo 'export PATH=/opt/gradle/gradle-8.13/bin:$PATH' >> ~/.bashrc

### Option B — SDKMAN

    curl -s "https://get.sdkman.io" | bash
    source "$HOME/.sdkman/bin/sdkman-init.sh"
    sdk install gradle 8.13

## After installing Gradle 8

    gradle --version   # must show 8.13
    gradle --stop      # kill any stale 7.4.2 daemons

    cd fortran-transpiler/FortranWeaver
    gradle installDist
